How to Test

For the best experience, open these mockups on an actual mobile device. Alternatively, use your browser's responsive design mode:

  • Chrome/Edge: Press F12, then Ctrl+Shift+M (Cmd+Shift+M on Mac)
  • Firefox: Press F12, then Ctrl+Shift+M (Cmd+Option+M on Mac)
  • Safari: Enable Develop menu, then choose "Enter Responsive Design Mode"

Try different device presets like iPhone 14 Pro, Pixel 7, or iPad Mini to see how the layouts adapt.
